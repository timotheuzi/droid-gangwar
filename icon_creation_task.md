# Icon Creation Task for Droid Gangwar Game

## Objective
Create comprehensive icons for the Droid Gangwar game including:
- Main game/app icon
- Icons for each room/location in the city
- Menu/navigation icons
- Action/feature icons

## Task Breakdown
- [ ] Analyze existing drawable resources and current icon structure
- [ ] Create main game/app icon with urban gang theme
- [ ] Create location/room icons:
  - [ ] Alleyway icon
  - [ ] Bank icon
  - [ ] Bar icon
  - [ ] Crackhouse icon
  - [ ] Gun Shack icon
  - [ ] Info Booth icon
  - [ ] Pick & Save store icon
  - [ ] City/Overview icon
- [ ] Create navigation and menu icons
- [ ] Create action/feature icons for gameplay
- [ ] Ensure all icons follow consistent visual style
- [ ] Test icons in the application context

## Visual Style Guidelines
- Urban/street aesthetic matching gang theme
- Consistent color scheme with existing gradients
- Clear, recognizable symbols for each location
- 24dp, 48dp, and 72dp versions for different screen densities
